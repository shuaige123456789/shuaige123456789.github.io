# 赛博朋克个人主页

炫酷赛博朋克风格个人网站模板，适合嵌入式/硬件/软件工程师。

## 🚀 部署到 GitHub Pages

已部署到：**https://shuaige123456789.github.io/**

### 推送更新

修改内容后，在项目目录运行：

```bash
git add .
git commit -m "更新内容"
git push
```

等 1-2 分钟，网站自动更新。

## ✏️ 自定义内容

打开 `index.html`，找到开头的 `CONFIG` 对象：

```js
const CONFIG = {
    name: 'Your Name',           // ← 改成你的名字
    skills: [ ... ],             // ← 改你的技能和熟练度
    projects: [ ... ],           // ← 改你的项目
};
```

其他需要修改的地方：
- 页面顶部的名字和标题
- "关于我"部分的介绍文字
- 联系方式（邮箱、地点）
- 页脚的社交链接和署名
